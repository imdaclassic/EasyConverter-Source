**HOW TO USE**

1. Open Start.bat
2. Select your gpu
3. Select a .pt or .onnx file
4. Select which type you want to convert to
5. Select FP32 or FP16
6. Wait for it to convert
7. When done model will be saved to original location

* Done! :) *
